# Plugins

Claude Code plugin marketplace (`claude-plugins-official`).

## Installing a plugin

```
claude plugin marketplace add davidshabani222-coder/Plugins
claude plugin install superpowers@claude-plugins-official
```

## Available plugins

- **superpowers** — Core skills library for Claude Code: TDD, debugging, collaboration patterns, and proven techniques. ([upstream](https://github.com/obra/superpowers))
